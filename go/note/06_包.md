# 简介：
> Go语言是用包(package)来组织代码的  
> 包是多个Go源码的集合，是一种高级的代码复用方案  
> 任何代码文件必须属于某个包  
> 源码文件的第一行有效代码必须是`package packageName`  
> 通过该语句声明自己所在的包  
> 一般包的名称就是其所在目录的名称  
---

# 包的习惯用法：
1. 包名一般小写，使用一个简短且有意义的名称  
2. 包名一般要和所在目录同名  
3. 包名为main的包应为程序的入口包  
4. 编译不含main包的源码文件不会得到可执行文件  
---

# 包的导入：
> ## 简介：
> 要在代码中引用其他包的内容，需要使用import关键字导入该包  
> 导入的包名需要用双引号阔起来  
> 包名是从GOPATH/src/后开始计算的，使用/进行路径分隔  
## 单行导入：
> ```go
> import 包名
> ```
## 多行导入：
> ```go
> import (
>  "包1路径"
>  "包2路径"
> )
> ```
